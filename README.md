# ConsoleTrigoC


	Console Trigonometry Solver in C
	(v0.1)	
	
	Note:
	*.Right-angled Triangle Only
	*.Only Two Values Are Considered For Evaluation.
	
	Solving Priority
	- A&B
	- B&C
	- A&C
	- X&A, Y&A
	- X&B, Y&B
	- X&C, Y&C
