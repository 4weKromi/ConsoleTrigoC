
# ConsoleTrigoC

## Console Trigonometry in C
### (v0.2)	
	
#### Note:
- Right-angled Triangle Only
- Only Two Values Are Considered For Evaluation.
	
#### Solving Priority
- A&B
- B&C
- A&C
- X&A, Y&A
- X&B, Y&B
- X&C, Y&C

---

### Build For Windows:
#### Requirement:
+ GNU GCC
+ GNU make/GCC mingw32-make

Source link : [src](https://github.com/4weKromi/ConsoleTrigoC/tree/master/source)
